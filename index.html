MOTDifier

Simple "Message of the Day" Script

Author:

Sebastian Kaiser kaiser.seb@gmail.com

LICENCE:

MIT

Description:

  • script as php or bash / shell script
  • for usage in crontab
  • php file will overwrite the /etc/motd file
  • shell script will output the motd - u have to pipe the output into /etc/motd
